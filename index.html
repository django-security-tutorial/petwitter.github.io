Prepare for the tutorial in 2 steps

1. Bring a computer

If possible, bring a computer with Python 3 and virtualenv installed.

If you don't know how to install virtualenv, please show up 15 minutes before the tutorial so we can help you.

2. Create an account on Zulip

By the time you see this page, you should have already received an invitation to the pettwitter Zulip organization. Join it!

If you haven't received one, that's OK. Send an email to: pettwitter@jacinda.io.
